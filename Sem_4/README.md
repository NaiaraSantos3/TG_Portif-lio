Desenvolver uma solução escalável e de alta disponibilidade para armazenar e consultar, em tempo real, dados de geolocalização gerados por dispositivos IoT.

## Tecnologias Utilizadas
As tecnologias utilizadas foram:

[Colocar a imagem das tecnologias]

![Tecnologias Utilizadas](Descricao_Projetos_Semestre/Imagens/tecnologias_4sem.png)

- Vue.js: framework JavaScript utilizado no frontend;
- Vuetify: biblioteca de componentes de interface com o _Vue.js_;
- Spring Boot: framework utilizado no backend para o desenvolvimento em Java;
- Redis: sistema de armazenamento em memória voltado para cache e manipulação eficiente de estruturas de dados, proporcionando uma resposta mais rápida e alta performance na aplicação.
- Docker: usado para unificar e gerenciar os ambientes de banco de dados, frontend e backend em contêineres;
- Oracle: banco de dados relacional utilizado pela sua escalabilidade e confiabilidade no armazenamento de informações;
- API Google Maps: utilizada para integração de mapas interativos, assegurando uniformidade no estilo e acesso a recursos como geolocalização, rotas e maior praticidade para uso da ferramenta;
- YouTrack: utilizada para organização de tarefas e visualização do progresso do time;

    - Criação de protótipos de telas, garantindo consistência visual, clareza nas funcionalidades e alinhamento com os requisitos do projeto.
